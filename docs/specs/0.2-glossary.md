# Orbion 术语表

| 术语 | 英文 | 定义 |
|------|------|------|
| **审批式执行** | Approval-based Execution | Agent 产出内容和执行计划后暂停，人类审批后才实际执行 |
| **接入端口对称** | Symmetric Entry Ports | SSEChannel 和 IMChannel 是 Event Bus 的两种对称接入方式，审批事件来自哪个通道对业务逻辑完全透明 |
| **Participant** | Participant | 统一人类和 Agent 的抽象参与者，在同一事件系统中平等交互 |
| **Event Bus** | Event Bus | 事件发布/订阅抽象接口，所有服务间通信的唯一通道 |
| **Event Store** | Event Store | 不可变事件日志，所有业务事件持久化存储的唯一写入口 |
| **CQRS** | CQRS | Command Query Responsibility Segregation — 读写分离：写端追加不可变事件，读端查询投影视图 |
| **投影** | Projection | CQRS 读端的可变聚合视图，可从 Event Store 完全重建 |
| **correlation_id** | correlation_id | 串联整个协作链的 UUID（"讨论→总结→分解→审批→执行"同一个 ID） |
| **causation_id** | causation_id | 记录直接因果关系的 UUID（触发当前事件的上级事件 ID） |
| **ChannelAdapter** | ChannelAdapter | Channel 插件抽象接口，SSEChannel 和 IMChannel 都实现此接口 |
| **SSEChannel** | SSEChannel | Web SSE 推送的 Channel 实现，将 EventBus 事件推送给前端 |
| **IMChannel** | IMChannel | IM 平台接入的 Channel 实现（Phase 2），翻译 IM 协议与 Orbion 事件 |
| **Hub Gateway** | Hub Gateway | 所有外部入口的统一基础设施枢纽，管入站、鉴权、Channel、EventBus、静态文件服务 |
| **ModelAdapter** | ModelAdapter | 模型适配器抽象接口，统一不同 LLM 的调用格式 |
| **Bitmask 权限位** | Bitmask Permission Bits | 单个 BIGINT 字段存储所有权限，高效且可扩展 |
| **三层覆盖** | Three-tier Override | Discord 风格权限覆盖：项目基础→空间级覆盖→资源级覆盖 |
| **Deny 胜过 Allow** | Deny Over Allow | 同层级权限冲突时 deny 生效 |
| **项目边界硬隔离** | Project Boundary Hard Isolation | 所有项目数据查询必须带 project_id |
| **ABAC** | ABAC | Attribute-Based Access Control — 上下文感知权限约束 |
| **AgentDeclaration** | AgentDeclaration | Agent 注册声明（role/goal/backstory/订阅事件/模型配置等） |
| **Skill** | Skill | Agent 的声明式工具能力，按风险分级（低/中/高） |
| **memory.md** | memory.md | Agent 层次化记忆文件，路径表达作用范围（平台/项目/Agent/任务四层） |
| **Context Injection** | Context Injection | 知识库内容序列化为结构化 prompt 注入 Agent 上下文 |
| **知识与记忆分离** | Knowledge-Memory Separation | 知识库管理私域知识（人类编写），Agent 记忆管理行为状态（机器生成），不混合存储 |
| **ExecutionPlan** | ExecutionPlan | 分解 Agent 产出的任务列表，人类审批后才交给执行 Agent |
| **TaskOutput** | TaskOutput | 执行 Agent 产出的代码/文档内容，人类审批后才 Git commit |
| **Thread** | Thread | 讨论线程，人类和 Agent 在同一消息流中交互的核心组织单元 |
| **Space** | Space | 权限空间，可覆盖项目基础权限 |
| **Saga 补偿** | Saga Compensation | Agent 失败时的自动补偿机制 |
| **自我构建** | Self-building | 用 Orbion 开发 Orbion 自己的验证目标 |
| **部分审批** | Partial Approval | 审批执行计划时可只批准部分任务 |
